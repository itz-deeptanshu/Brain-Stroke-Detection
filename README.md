# 🧠 Brain Stroke Prediction — End-to-End ML Pipeline

A complete machine learning project to predict the likelihood of a brain stroke in patients using clinical and demographic features. Built with Python, scikit-learn, XGBoost, and SMOTE to handle severe class imbalance.

---

## 📌 Dataset

**Source:** [Stroke Prediction Dataset — Kaggle (fedesoriano)](https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset)

**Download:**
```bash
kaggle datasets download -d fedesoriano/stroke-prediction-dataset
unzip stroke-prediction-dataset.zip
```

**Shape:** 5,110 rows × 12 columns

| Column | Type | Description |
|---|---|---|
| `id` | int | Patient identifier (dropped) |
| `gender` | cat | Male / Female / Other |
| `age` | float | Patient age |
| `hypertension` | int | 0 = no, 1 = yes |
| `heart_disease` | int | 0 = no, 1 = yes |
| `ever_married` | cat | Yes / No |
| `work_type` | cat | Private / Self-employed / Govt_job / children / Never_worked |
| `Residence_type` | cat | Urban / Rural |
| `avg_glucose_level` | float | Average blood glucose (mg/dL) |
| `bmi` | float | Body Mass Index (3.93% missing) |
| `smoking_status` | cat | never smoked / formerly smoked / smokes / Unknown |
| `stroke` ⭐ | int | **Target** — 0 = No Stroke, 1 = Stroke |

**Class Imbalance:** Only ~4.87% of patients had a stroke → SMOTE applied on training set.

## 🧹 Data Cleaning Summary

| Issue | Treatment |
|---|---|
| `id` column | Dropped (no predictive value) |
| `gender = 'Other'` | Removed (only 1 row) |
| Missing `bmi` (3.93%) | Median imputation (right-skewed distribution) |
| Outliers in `age`, `bmi`, `avg_glucose_level` | IQR Winsorization (capping) |
| Duplicate rows | Checked and removed if any |

---

## 🔬 EDA Highlights

- **Age** is the single strongest predictor — stroke rate jumps from ~0.4% (<20) to ~8.3% (65+)
- **Glucose level** — diabetic range (>125 mg/dL) shows 2× higher stroke rate
- **Hypertension + Heart Disease** — each roughly doubles individual risk
- **Smoking** — "formerly smoked" shows unexpectedly higher rate than "smokes" (survivor bias)
- **BMI** — mild positive correlation; obesity alone is a weaker predictor than age or glucose

## 📊 Evaluation Metrics

All models are evaluated on the **held-out test set (20%)** using:

- **Accuracy** — overall correct predictions
- **Precision** — of predicted stroke cases, how many are real?
- **Recall** — of actual stroke cases, how many did we catch? *(most important for medical use)*
- **F1-Score** — harmonic mean of Precision & Recall
- **ROC-AUC** — ranking quality across all thresholds
- **5-Fold Stratified CV ROC-AUC** — stability check

> ⚕️ **Medical note:** In stroke prediction, **Recall (sensitivity) is critical** — missing a true stroke is far more dangerous than a false alarm. Consider lowering the threshold from 0.5 to 0.35 in production to maximise recall.

